## 📋 项目简介

一个轻量级的个人内容分析工具，帮助内容创作者整理和分析小红书公开笔记，用于选题参考研究。

### 核心功能

1. **笔记信息录入** — 手动输入或粘贴小红书公开笔记链接，通过表单提交笔记信息
2. **数据存储** — SQLite 本地存储，支持按多种维度排序和检索
3. **内容分析** — 高频关键词提取、内容类型分类、互动数据分析、爆款共性总结
4. **可视化展示** — 基于 Streamlit 的 Web 界面，包含 Dashboard、笔记列表、分析页
5. **AI 总结** — 本地规则版笔记分析 + 大模型 API 接口预留

### 技术栈

| 技术 | 用途 |
|------|------|
| Python 3.8+ | 主语言 |
| Streamlit | Web 界面 |
| SQLite | 数据存储 |
| pandas | 数据处理 |
| jieba | 中文分词 & 关键词提取 |
| plotly | 交互式图表 |

## 🚀 快速开始

```bash
# 克隆仓库
git clone <repo-url>
cd xhs_note_analyzer

# 安装依赖
pip install -r requirements.txt

# 初始化数据库并插入示例数据
python seed_data.py

# 启动 Web 界面
streamlit run app.py
```

启动后浏览器打开 `http://localhost:8501` 即可使用。

## 🗂️ 页面说明

### 1. 首页 Dashboard
- 已录入笔记总数 / 总点赞数 / 总收藏数 / 平均互动数
- 互动最高的前 5 篇笔记卡片

### 2. 新增笔记
- 表单字段：链接、标题、正文、作者、点赞/收藏/评论数、发布时间、内容类型、标签、备注
- 提交后写入 SQLite，支持连续录入

### 3. 笔记列表
- 搜索标题/正文
- 按点赞/收藏/评论排序
- 按内容类型筛选
- 点击查看详情

### 4. 分析页
- 高频关键词 Top 20（柱状图）
- 内容类型分布（饼图）
- 互动数据排行榜
- 爆款笔记共性总结
- 选题方向建议

### 5. 笔记详情
- 完整笔记信息
- AI 分析结果（本地规则版）
